## **💡 How to Run?**  
1. Clone this repository:  
   ```sh
   git clone https://github.com/lalitpatil891/codewith-react.git
   ```
2. Navigate to the folder:  
   ```sh
   cd codewith-react
   ```
3. Install dependencies:  
   ```sh
   npm install
   ```
4. Start the development server:  
   ```sh
   npm run dev
   ```
5. Open in browser at: **http://localhost:5173/**
